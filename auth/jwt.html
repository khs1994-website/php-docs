  2. 服务器验证 username password

  3. 获取用户信息，生成 token，返回 token

  4. token 存入 cookie

认证过程

  1. 请求头中加入 Authorization: Bearer YOUR_JWT

  2. 服务器解密 token

  3. 验证过期时间，验证权限

  4. 返回结果
