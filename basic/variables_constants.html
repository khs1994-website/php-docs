静态变量

静态变量仅在局部函数域中存在，但当程序执行离开此作用域时，其值并不丢失。

function test()
{
    static $a = 0;
    echo $a;
    $a++;
}

$GLOBALS 数组

可变变量

$a='hello';

$$a='world'; // 即 $hello='world';

常量

常量的范围是全局的，常量一旦定义就不能被重新定义或者取消定义。

常量的值只能是标量。

定义常量

define('FOO', 'bar');

const FOO = 'bar';

const 不能用在函数、循环、if 语句之内。

获取常量

  • constant() 获取常量值。

  • get_define_constants() 获得所有已定义的常量。
