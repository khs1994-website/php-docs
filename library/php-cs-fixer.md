```bash
$ curl -L https://cs.sensiolabs.org/download/php-cs-fixer-v2.phar -o php-cs-fixer

$ sudo mv php-cs-fixer /usr/local/bin

$ sudo chmod +x /usr/local/bin/php-cs-fixer
```

Edit `.php_cs`

```bash
$ php-cs-fixer fix
```
